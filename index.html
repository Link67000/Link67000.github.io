presentation of my project :

  • This project follows on from courses on HTML and CSS, which are programming languages used in the creation of web pages. To give a quick overview of these two languages: HTML stands for "HyperText Markup Language" and is used to structure a web page and its content. CSS stands for "Cascading Style Sheets" and is designed to complement HTML and give it a customised style.
  • During our first year of study, we will be learning to use these web programming languages. This will be useful to a greater or lesser extent depending on where we go next, but it may also prove useful for personal projects or working in a company.
  • This project completes our introduction to WEB languages: it's designed to allow us to confront all the problems encountered by web designers. The aim is to put to the test the skills we have learnt in class.
